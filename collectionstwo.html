Android-Focused Kotlin Collections

1️⃣ Why are Kotlin collections preferred in Android?

Explain why Kotlin collections are commonly used in Android development.
They are type-safe, concise, immutable by default, reduce boilerplate, and work seamlessly with Android APIs.
---

2️⃣ Difference between List and MutableList in Android?

Which should be used in RecyclerView?
Use List for exposing data to UI and MutableList internally for updates. 
private val items = mutableListOf()
fun getItems(): List = items

5️⃣ What is DiffUtil and how collections help?

Why immutable lists are recommended with DiffUtil?
DiffUtil compares old and new lists efficiently. Immutable lists prevent unexpected mutations.

9️⃣ What is a Sequence and when to use it?

Why sequences are useful for large datasets?
Sequences use lazy evaluation, improving performance and memory usage. 
items.asSequence()
     .filter { it.isActive }
     .map { it.name }
     .toList()
