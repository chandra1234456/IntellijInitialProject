📱 Android Interview Questions

🧠 Activity Lifecycle

Q1. What are Activity lifecycle methods?

onCreate → onStart → onResume → onPause → onStop → onDestroy

Q2. When is onPause() called?

When activity is partially visible or another activity comes in front.

Q3. Difference between onStop() and onDestroy()?

onStop → activity not visible
onDestroy → activity is finishing or destroyed

Q4. What is onSaveInstanceState()?

Used to save UI state before destruction.

🧩 Fragment Lifecycle

Q1. Fragment lifecycle flow?

onAttach → onCreate → onCreateView → onViewCreated → onStart → onResume

Q2. Difference between Activity & Fragment lifecycle?

Fragment has additional view lifecycle methods.

Q3. onCreateView vs onViewCreated?

onCreateView → inflate layout
onViewCreated → initialize UI

Q4. Why use Fragment?

  • Reusable UI
  • Better modularity

🔗 Intents

Q1. What is an Intent?

Messaging object used to communicate between components.

Q2. Types of Intent?

  • Explicit → target component specified
  • Implicit → system decides component

Q3. Example of Explicit Intent

Intent(this, SecondActivity::class.java)

Q4. Example of Implicit Intent

Intent(Intent.ACTION_VIEW, Uri.parse("https://google.com"))

Q5. What is Intent Filter?

Defines which intents an app can respond to.

🔄 Tasks & Back Stack

Q1. What is Back Stack?

A stack of activities managed by Android.

Q2. What happens when back button is pressed?

Current activity is popped and previous is resumed.

Q3. What is launchMode?

  • standard
  • singleTop
  • singleTask
  • singleInstance

Q4. What is taskAffinity?

Defines which task an activity belongs to.

🌍 Context

Q1. What is Context?

Provides access to application resources and system services.

Q2. Types of Context?

  • Application Context
  • Activity Context

Q3. Difference between Application & Activity Context?

Application → global, long-lived
Activity → tied to UI lifecycle

Q4. When to use Application Context?

  • Singletons
  • Database

Q5. Memory leak example?

Holding Activity context in static variable.
