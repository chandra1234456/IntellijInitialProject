Kotlin Practice Problems

String Problems

  1. Reverse a string.
    Example: Input "Kotlin" → Output "niltoK"
  2. Check if a string is a palindrome.
    Example: Input "madam" → Output true
  3. Count the number of vowels in a string.
    Example: Input "Hello World" → Output 3
  4. Count total characters in a string excluding spaces.
    Example: Input "Hello Kotlin" → Output 11
  5. Convert a string to uppercase and lowercase.
  6. Find the first non-repeating character in a string.
    Example: Input "swiss" → Output 'w'
  7. Remove all whitespaces from a string.
    Example: Input "Kotlin is fun" → Output "Kotlinisfun"
  8. Replace all occurrences of a character with another.
    Example: Input "banana", replace 'a' with 'o' → Output "bonono"
  9. Count occurrences of a character in a string.
    Example: Input "programming", character 'g' → Output 2
  10. Check if two strings are anagrams.
    Example: Input "listen" & "silent" → Output true

Sorting Problems

  1. Sort an array of integers in ascending order.
    Example: Input [5, 2, 9, 1, 5] → Output [1, 2, 5, 5, 9]
  2. Sort an array of strings alphabetically.
    Example: Input ["Kotlin", "Java", "Python", "C++"] → Output ["C++", "Java", "Kotlin", "Python"]
  3. Sort an array of integers in descending order.
    Example: Input [3, 8, 1, 6] → Output [8, 6, 3, 1]
  4. Sort a list of strings by length (shortest to longest).
    Example: Input ["apple", "kiwi", "banana", "fig"] → Output ["fig", "kiwi", "apple", "banana"]
  5. Sort characters in a string alphabetically.
    Example: Input "kotlin" → Output "iklnot"
  6. Sort numbers based on the number of digits.
    Example: Input [10, 2, 300, 45] → Output [2, 10, 45, 300]
  7. Sort even numbers ascending and odd numbers descending in an array.
    Example: Input [5, 2, 8, 3, 1] → Output [2, 8, 5, 3, 1]
  8. Find the top K the largest elements in an array.
    Example: Input [4, 1, 7, 3, 9], K=3 → Output [7, 9, 4]
  9. Sort a list of objects by a property.
    Example: Students with marks: [("Alice",85),("Bob",90),("Carol",80)] → Sorted by marks ascending: [("Carol",80),("Alice",85),("Bob",90)]
  10. Sort a 2D array row-wise or column-wise.
    Example: Input [[3,1],[2,4]] → Row-wise output: [[1,3],[2,4]]

Collections Problems

  1. Create a list of integers and find the sum of all elements.
    Example: Input [1,2,3,4] → Output 10
  2. Find the largest and smallest element in a list.
    Example: Input [10, 5, 20, 8] → Output: Max=20, Min=5
  3. Count the frequency of each element in a list.
    Example: Input [1,2,2,3,1] → Output: {1=2, 2=2, 3=1}
  4. Remove duplicate elements from a list.
    Example: Input [1,2,2,3,1] → Output [1,2,3]
  5. Check if a list contains a specific element.
    Example: Input [1,2,3,4], check 3 → Output true
  6. Merge two lists into one and remove duplicates.
    Example: Input [1,2,3] + [2,3,4] → Output [1,2,3,4]
  7. Sort a map by keys and by values.
    Example: Input {"b":2,"a":1,"c":3} → Sorted by keys: {"a":1,"b":2,"c":3}
  8. Filter a list to only include even numbers.
    Example: Input [1,2,3,4,5] → Output [2,4]
  9. Group a list of strings by their first character.
    Example: Input ["apple","banana","avocado"] → Output {'a': ["apple","avocado"], 'b': ["banana"]}
  10. Check if two lists are equal (contain the same elements).
    Example: Input [1,2,3] & [3,2,1] → Output false

Swapping Problems

  1. Swap two integers without using a temporary variable.
    Example: Input a=5, b=10 → Output a=10, b=5
  2. Swap two strings without using a temporary variable.
    Example: Input x="Hello", y="World" → Output x="World", y="Hello"
  3. Swap two elements in an array given their indices.
  4. Swap first and last element of an array.
  5. Swap adjacent elements of an array (pairwise).
