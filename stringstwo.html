Kotlin Interview Questions

🔤 Kotlin String Questions

1. Check if string contains only digits:
str.all { it.isDigit() }

2. Difference between == and ===:
== checks structural equality (content).
=== checks referential equality (same object).

3. Reverse a string:
str.reversed()

4. Output of "Kotlin".substring(1, 4):
Output: "otl"

5. Remove all whitespace:
str.replace("\\s".toRegex(), "")

6. Capitalize first letter:
str.replaceFirstChar { it.uppercase() }

7. Split by multiple delimiters:
str.split(",", ";", " ")

8. Calling trim() on null:
Throws NullPointerException. Use safe call:
str?.trim()

9. toCharArray() vs iterator():
toCharArray() returns array of characters.
iterator() provides sequential access.

10. String formatting:
String.format("Hello %s", name)
or
"Hello $name"

📋 Kotlin List Questions

1. listOf() vs mutableListOf():
listOf() → immutable list.
mutableListOf() → mutable list.

2. Remove duplicates:
list.distinct()

3. Output of listOf(1,2,3).drop(1):
[2, 3]

4. Filter null values:
list.filterNotNull()

5. map() vs flatMap():
map() transforms elements.
flatMap() flattens nested collections.

6. Second largest number:
list.distinct().sortedDescending()[1]

7. List vs MutableList vs ArrayList:
List → read-only interface.
MutableList → allows modification.
ArrayList → resizable array implementation.

8. Check if all elements satisfy condition:
list.all { it > 0 }

9. Group elements by key:
list.groupBy { it.length }

10. Sorting a list:
list.sorted()
list.sortedDescending()
list.sortedBy { it.property }
