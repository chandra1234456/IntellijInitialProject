RecyclerView & Pagination Interview Q&A

📌 RecyclerView Basics

What is RecyclerView?
RecyclerView is a flexible and efficient view for displaying large datasets by recycling item views.
What are main components of RecyclerView?
Adapter, ViewHolder, LayoutManager, and ItemDecoration.
What is ViewHolder pattern?
It stores view references to avoid repeated findViewById calls, improving performance.
Difference between ListView and RecyclerView?
RecyclerView is more flexible, supports multiple layouts, animations, and uses ViewHolder by default.

🚀 RecyclerView Optimization

How do you optimize RecyclerView performance?
Use ViewHolder efficiently, enable setHasFixedSize(true), use DiffUtil, avoid nested layouts, and use pagination for large data.
What is DiffUtil?
DiffUtil calculates differences between two lists and updates only changed items instead of refreshing entire list.
What is setHasFixedSize(true)?
It improves performance when RecyclerView size doesn’t change, avoiding unnecessary layout calculations.
How to avoid lag in RecyclerView?
Use lightweight layouts, avoid heavy operations in onBindViewHolder, load images with libraries like Glide, and use background threads.
What causes RecyclerView jank?
Large images, complex layouts, frequent notifyDataSetChanged calls, and doing heavy work on main thread.

📄 Pagination

What is pagination?
Pagination is loading data in chunks instead of all at once to improve performance and reduce memory usage.
How do you implement pagination in RecyclerView?
Add scroll listener to detect end of list and load next page, or use Paging 3 library for automatic pagination.
What is Paging 3 library?
It is a Jetpack library that helps load paginated data from network or database efficiently using Flow and coroutines.
Difference between manual pagination and Paging 3?
Manual pagination requires handling scroll and API calls manually, while Paging 3 handles caching, loading states, and errors automatically.
What is LoadState in Paging?
LoadState represents loading, success, or error states during pagination.

💡 Scenario-Based Questions

RecyclerView scrolling is lagging. What will you do?
Optimize layouts, use DiffUtil, reduce overdraw, compress images, and move heavy work off the main thread.
How do you handle duplicate data in pagination?
Use unique IDs, filter duplicates, or rely on Paging 3 which handles it internally.
How do you show loading in pagination?
Add a loading item at the end of list or use Paging LoadStateAdapter.
