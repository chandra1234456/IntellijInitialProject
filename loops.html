For Loops in Kotlin (With Use Cases)

1. Basic Range Loop

Use Case: Printing numbers, generating IDs, running a task fixed number of times.

This loop runs from 1 to 5 (inclusive).

for (i in 1..5) {
    println(i)
}

2. Loop with until (Exclusive End)

Use Case: Iterating array indexes where last index should be excluded.

Stops before reaching 5.

for (i in 0 until 5) {
    println(i)
}

3. Loop with Step

Use Case: Skipping values like printing odd numbers or sampling data.

Increments by 2 each time.

for (i in 1..10 step 2) {
    println(i)
}

4. Reverse Loop (downTo)

Use Case: Countdown timers, reverse traversal.

Counts backwards from 10 to 1.

for (i in 10 downTo 1) {
    println(i)
}

5. Loop Through an Array or List

Use Case: Reading user data, processing lists of values.

Iterates through all elements in a list.

val numbers = listOf(10, 20, 30)

for (num in numbers) {
    println(num)
}

6. Loop with Index

Use Case: When both position and value are required (e.g., ranking systems).

Accesses elements using index.

for (i in numbers.indices) {
    println("Index $i = ${numbers[i]}")
}

Using withIndex()

More readable and preferred Kotlin style.

for ((index, value) in numbers.withIndex()) {
    println("Index $index = $value")
}

7. Loop Through a String

Use Case: Character validation, password checks, text processing.

Each character is accessed one by one.

for (ch in "Kotlin") {
    println(ch)
}

8. Nested For Loops

Use Case: Tables, matrix operations, comparing combinations.

One loop inside another.

for (i in 1..3) {
    for (j in 1..2) {
        println("i=$i, j=$j")
    }
}

9. Using repeat()

Use Case: Running a task multiple times without needing an index.

Executes block 5 times.

repeat(5) {
    println("Hello")
}

10. Break and Continue

Use Case: Skipping unwanted values or stopping early when a condition is met.

break → stops loop
continue → skips current iteration 

for (i in 1..10) {
    if (i == 5) break
    if (i % 2 == 0) continue
    println(i)
}
