Time & Space Complexity - O(1) Explained

What is Time Complexity?

Time Complexity tells us how the running time increases as input size (n) increases.

  • We ignore actual seconds.
  • We ignore system speed.
  • We focus on growth rate.

O(1) Time Complexity

O(1) means constant time. The operation takes the same time regardless of input size.

Example 1: Accessing Array Element

val arr = intArrayOf(10, 20, 30, 40)
println(arr[2])

Direct index access takes constant time → O(1)

Example 2: Simple Assignment

var x = 5

Single operation → O(1)

Example 3: Swap Two Numbers

var a = 5
var b = 10
val temp = a
a = b
b = temp

Fixed number of steps → O(1)

If there are no loops or recursion, it is usually O(1).

O(1) Space Complexity

O(1) Space means constant extra memory usage regardless of input size.

Example 1: Find Maximum

fun findMax(arr: IntArray): Int {
    var max = arr[0]
    for (i in arr.indices) {
        if (arr[i] > max) {
            max = arr[i]
        }
    }
    return max
}

Time = O(n) but Space = O(1) because only one variable is used.

Example 2: Reverse Array In-Place

fun reverse(arr: IntArray) {
    var start = 0
    var end = arr.size - 1

    while (start < end) {
        val temp = arr[start]
        arr[start] = arr[end]
        arr[end] = temp
        start++
        end--
    }
}

No new array created → Space = O(1)

Example of NOT O(1) Space

val newArr = IntArray(n)

Creates new array of size n → Space = O(n)

Quick Rules to Identify O(1)

Time is O(1) if:

  • No loops
  • No recursion
  • No input-size dependent operations
  • Direct access

Space is O(1) if:

  • No new arrays created
  • No new lists created
  • Only fixed number of variables used

Summary Table

Code Type Time Complexity Space Complexity
Array index access O(1) O(1)
Simple swap O(1) O(1)
Single loop O(n) O(1)
Nested loop O(n²) O(1)
Create new array of size n O(n) O(n)

Interview Tip

Very common interview question: "Can you optimize space to O(1)?"

That means: Do not create extra arrays. Modify original data structure.
