Common String Problems

Problem 1: Reverse a string in Android.
You can use StringBuilder("text").reverse().toString() or a manual loop.
Problem 2: Check if a string is a palindrome.
Compare the string with its reverse. If they are equal, it's a palindrome.
Problem 3: Count the number of vowels and consonants in a string.
Loop through the string and check if each character is a vowel or consonant.
Problem 4: Remove all whitespaces from a string.
Use str.replaceAll("\\s+", "").
Problem 5: Find the first non-repeating character in a string.
Use a HashMap to count character occurrences and return the first with count 1.
Problem 6: Check if two strings are anagrams.
Sort both strings and compare them, or use a frequency array to count characters.
Problem 7: Convert a string to uppercase and lowercase.
Use str.toUpperCase() and str.toLowerCase().
Problem 8: Remove duplicate characters from a string.
Use a LinkedHashSet to maintain order and remove duplicates.
Problem 9: Count occurrences of a substring in a string.
Loop with indexOf() and count matches, or use regex.
Problem 10: Check if a string contains only digits.
Use str.matches("\\d+").
