Android CI/CD Interview Questions & Answers

Basic

What is CI/CD?
CI/CD stands for Continuous Integration and Continuous Delivery/Deployment. It automates building, testing, and releasing apps to ensure faster and reliable delivery.
Difference between CI, CD, and Deployment?
CI integrates code changes frequently. Continuous Delivery ensures code is always ready to release. Continuous Deployment automatically releases code to production.
What is Gradle?
Gradle is the build system used in Android to compile code, manage dependencies, and generate APK/AAB files.

Intermediate

How do you automate Android builds?
Using tools like Jenkins or GitHub Actions, configure pipelines to run Gradle commands (e.g., ./gradlew assembleDebug) on code push.
How do you manage secrets?
Store secrets in CI tools (environment variables or secret managers) instead of hardcoding them in code.
What is signing config?
Signing config is used to sign Android apps with a keystore before releasing them to the Play Store.
How do you run tests in CI?
Run unit tests using ./gradlew test and UI tests using ./gradlew connectedAndroidTest or Firebase Test Lab.

Advanced

How do you optimize build times?
Use Gradle caching, parallel builds, build variants optimization, and avoid unnecessary dependencies.
What is caching in CI/CD?
Caching stores dependencies and build outputs to avoid re-downloading or rebuilding, improving speed.
How do you deploy to Play Store?
Use tools like Fastlane or Google Play Developer API to automate APK/AAB upload and release.

Scenario-Based

How to fix a failing build?
Check logs, identify errors, reproduce locally, fix issues, and re-run pipeline.
How do you handle flaky tests?
Stabilize tests, add retries, isolate failures, and mock dependencies.
How do you secure keystore files?
Encrypt and store them securely in CI tools or secret storage, not in the repository.

Tool-Specific

What is Fastlane?
Fastlane automates build, test, and deployment tasks like uploading apps to Play Store.
Role of Docker?
Docker ensures consistent build environments across machines in CI/CD pipelines.
What is Firebase App Distribution?
It allows distributing pre-release Android apps to testers easily during CI/CD.
