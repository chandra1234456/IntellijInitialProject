Java Collections – Interview & Preparation Questions

List Based Questions

1. Given a List containing [10, 20], apply swap on index 0 and 1.
Output: [20, 10]
2. Add elements 1,2,3,4 to ArrayList. Remove element at index 2.
Output: [1, 2, 4]
3. Sort List [5,3,1,4,2] in ascending order.
Output: [1, 2, 3, 4, 5]
4. Reverse List [1,2,3,4].
Output: [4, 3, 2, 1]
5. Find max element in [7,2,9,4].
Output: 9
6. Find frequency of 5 in [5,1,5,2,5].
Output: 3
7. Rotate list [1,2,3,4,5] by 2 positions right.
Output: [4, 5, 1, 2, 3]
8. Shuffle [1,2,3,4]. (Assume fixed shuffle result)
Output: [3, 1, 4, 2]
9. Convert array [10,20,30] to List and print size.
Output: 3
10. Check if list [1,2,3] contains 4.
Output: false

Set Based Questions

11. Add 1,2,2,3 to HashSet.
Output: [1, 2, 3]
12. Insert elements into TreeSet [5,1,4].
Output: [1, 4, 5]
13. Find union of sets {1,2,3} and {3,4,5}.
Output: [1, 2, 3, 4, 5]
14. Find intersection of sets {1,2,3} and {2,3,4}.
Output: [2, 3]
15. Check if LinkedHashSet maintains insertion order for [3,1,2].
Output: [3, 1, 2]

Map Based Questions

16. Insert (1,A), (2,B), (1,C) into HashMap.
Output: {1=C, 2=B}
17. Retrieve value for key 2 from {1=A,2=B,3=C}.
Output: B
18. Iterate over LinkedHashMap with entries inserted as 3,C 1,A 2,B.
Output: 3=C, 1=A, 2=B
19. Insert 3,1,2 into TreeMap.
Output: {1=..., 2=..., 3=...}
20. Count frequency of characters in "aabbc".
Output: {a=2, b=2, c=1}

Queue & Deque Questions

21. Offer 10,20,30 into Queue and poll once.
Output: 10 removed, Queue: [20, 30]
22. Use PriorityQueue with elements 5,1,3.
Output: Poll order → 1, 3, 5
23. AddFirst 1, AddLast 2 into Deque.
Output: [1, 2]

Conceptual & Advanced Questions

24. What happens when modifying ArrayList during iteration?
Output: ConcurrentModificationException
25. Difference between Comparable and Comparator sorting.
Output: Comparable → natural order; Comparator → custom order
26. Time complexity of get() in ArrayList.
Output: O(1)
27. Time complexity of search in HashSet (average case).
Output: O(1)
28. Convert List [1,2,3] to synchronized list.
Output: Thread-safe List
29. Use Collections.min on [4,8,2,6].
Output: 2
30. Swap elements at index 1 and 3 in [10,20,30,40].
Output: [10, 40, 30, 20]
