Data Structures & Algorithms (DSA) - Basic Problems

1. Arrays & Strings

Goal: Learn indexing, iteration, and simple manipulation.

  • Reverse an array
  • Find the maximum and minimum in an array
  • Sum of all elements in an array
  • Remove duplicates from an array
  • Reverse a string
  • Check if a string is a palindrome
  • Count vowels in a string

2. Linked Lists

Goal: Understand nodes, pointers, and dynamic memory.

  • Implement a singly linked list
  • Insert a node at the beginning, middle, and end
  • Delete a node
  • Find length of a linked list
  • Reverse a linked list
  • Detect a cycle in a linked list

3. Stacks & Queues

Goal: Learn LIFO (stack) and FIFO (queue) concepts.

  • Implement a stack using array or linked list
  • Implement a queue using array or linked list
  • Check for balanced parentheses in a string
  • Reverse a stack using recursion
  • Implement a circular queue

4. Recursion

Goal: Understand how functions can call themselves.

  • Factorial of a number
  • Fibonacci series (recursive and iterative)
  • Sum of array elements recursively
  • Tower of Hanoi
  • Print all subsets of a set

5. Searching & Sorting

Goal: Learn how to efficiently find and order elements.

  • Linear search
  • Binary search (on sorted arrays)
  • Bubble sort
  • Selection sort
  • Insertion sort
  • Count occurrences of a number in a sorted array

6. Hashing / Hash Tables

Goal: Understand fast lookups and frequency counting.

  • Count occurrences of elements in an array
  • Check if a string has all unique characters
  • Two-sum problem
  • Intersection of two arrays

7. Basic Trees

Goal: Learn hierarchical data storage.

  • Implement a binary tree
  • Preorder, Inorder, Postorder traversal
  • Find height of a tree
  • Count total nodes
  • Check if a tree is a BST
